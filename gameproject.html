Introduction

This paper talks about a project i am going to work on.
I wanted to test the skills that i had learned in Python, C# and machine learning so i came up with the idea of making a game where all the artwork and music is generated by an adveserial neural network.
I have not seen anyone do this before so i thought i might give it a shot and see if it works out.

I am hoping this project will teach me a lot about machine learning, Python and C#.
