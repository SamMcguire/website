Electro Magnet

I built a controlled electro magnet that makes a magnet float using Arduino and some electronic parts. I used a hall magnet sensor to see how far the magnet was away and then wired it to the Arduino board. I then wrote a quick code to control the power of the electromagnet depending in how far the magnet was below. This approach did not work because the box I had it attached to moved too much to be able to accurately measure how far away the magnet was.
